# RFM-Customer-Segmentation-Analysis-Python-
This project applies RFM Analysis (Recency–Frequency–Monetary) to segment customers based on their purchasing behavior. The goal is to identify high-value customers, at-risk segments, and insights that support data-driven marketing and retention strategies.

Data Cleaning & Preprocessing

RFM Metric Calculation (Recency, Frequency, Monetary)

Quantile-based RFM Scoring

Customer Segmentation Logic

Interactive Data Visualization (Plotly)

Business Insight Generation


Python

Pandas

NumPy

Plotly

Jupyter/Colab


**RFM Calculation Summary:**

Recency: Days since last purchase

Frequency: Number of transactions

Monetary: Total revenue generated

Scores (1–5) assigned using quantiles to classify customers.
